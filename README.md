Project Structure-
In src there are- 
  components folder which contains main folders like Header , VideoPlayer, VideoPlayerList 
  images folder which contains images used all over the app 
  shared folder which contains the components which are comman and can be re-used in other components like loader .
  constant folder contains the constant data which can be used throughout the app 
  customHook which contains the device info
